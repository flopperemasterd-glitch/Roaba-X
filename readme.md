# 🛞 ROABA X 🛞
### "o roata, hackuri infinite" 
*(One wheel, infinite hacks)*

A Romanian-themed memory editor and process manipulation tool written in C#.

## Features
-  Memory reading
-  Memory writing  
-  Value scanning
-  Pattern scanning (AOB with wildcards)
-  Advanced mode with function unprotect etc
-  Bonus: Spinning ASCII cube

## What is a Roaba?
A **roaba** is a wheelbarrow in Romanian. This tool is as stable as a wheelbarrow with one wheel - it works, but it might tip over at any moment.

## Installation
1. Download the source
2. Compile with Visual Studio or `csc Program.cs`
3. Run as **Administrator** (important!)

## Usage
```bash
RoabaX.exe
```

Select a process, then choose from:
- Read memory at address
- Write memory (integers, floats, bytes, or inject DLL)
- Scan for specific value
- ROABA TURBO
- Pattern scan with wildcards (e.g., `48 8B 05 ?? ?? ?? ??`)
